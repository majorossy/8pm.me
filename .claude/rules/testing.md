# Testing Defaults

When testing any feature in this project, always use **Railroad Earth** as the default test artist unless otherwise specified.

## Examples

- Import testing: `bin/magento archive:download "Railroad Earth"`
- GraphQL queries: Use Railroad Earth category/products
- UI testing: Navigate to Railroad Earth artist page
- Search testing: Search for "Railroad Earth" tracks/shows
- Populate testing: `bin/magento archive:populate "Railroad Earth"`

## Why Railroad Earth?

- Medium-sized catalog (not too large like Grateful Dead, not too small)
- Good variety of shows and recordings
- Consistent metadata quality from Archive.org
